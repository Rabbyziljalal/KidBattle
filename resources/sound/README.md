# Sound Effects Guide

This folder should contain the following sound files for the game:

## Required Files:

1. **correct.mp3**
   - Played when a player answers correctly
   - Suggested: Happy chime, ding, or success sound
   - Duration: 0.5-2 seconds

2. **wrong.mp3**
   - Played when a player answers incorrectly
   - Suggested: Buzzer, error tone, or gentle "oops" sound
   - Duration: 0.5-2 seconds

3. **gameover.mp3**
   - Played when the game ends
   - Suggested: Celebration music, victory fanfare, or final whistle
   - Duration: 2-5 seconds

## Free Sound Resources:

- **Freesound.org** - https://freesound.org/
  - Search: "success sound", "error buzz", "game over"
  
- **Zapsplat.com** - https://www.zapsplat.com/
  - Category: Game Sounds > UI Sounds
  
- **Mixkit.co** - https://mixkit.co/free-sound-effects/
  - Category: Game > Success/Fail sounds

## Creating Your Own:

You can also use online tools to generate simple sounds:
- **Beepbox.co** - Create custom chiptune sounds
- **SFXR** - Generate retro game sounds
- **Audacity** - Free audio editor to create/edit sounds

## Important Notes:

- All sound files should be in MP3 format
- Keep file sizes small (under 100KB each)
- Ensure sounds are kid-friendly and not too loud
- The game will work perfectly without sound files!

## Installation:

1. Download or create your sound files
2. Rename them exactly as: correct.mp3, wrong.mp3, gameover.mp3
3. Place them in this folder (resources/sound/)
4. Refresh your game in the browser

That's it! Your game will now have sound effects.

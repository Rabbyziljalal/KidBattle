# Team Name Customization Guide

## Overview
Brain Tug now supports **fully customizable team names**! You can create personalized team-based competitions that make the game more engaging for your specific group of players.

## How to Customize Team Names

### Step 1: Launch the Game
Open `index.html` in your web browser.

### Step 2: Enter Team Names
On the difficulty selection screen, you'll see two input fields:
- **🔵 Team 1 Name:** Enter the name for the first team
- **🔴 Team 2 Name:** Enter the name for the second team

### Step 3: Choose Difficulty
Select the appropriate difficulty level (Easy, Medium, or Hard).

### Step 4: Start Playing!
Your custom team names will appear throughout the game:
- Score section headers
- Arena labels (on both sides of the rope)
- Turn indicator ("LIONS's Turn")
- Winner announcement ("🎉 TIGERS Wins! 🎉")
- Final score screen

## Default Behavior

If you **don't enter** any team names, the game uses the default values:
- **TEAM A** (Blue team)
- **TEAM B** (Red team)

## Changing Team Names Mid-Game

### During a Game:
1. Click **"🔧 Change Level"** button
2. Confirm you want to leave the current game
3. Enter new team names
4. Select difficulty and start a new game

### After Game Over:
1. Click **"🎮 Play Again"** button
2. Click **"🔧 Change Level"** to go back to team selection
3. Update team names as desired
4. Start a new game

## Team Names Persist

Once you set team names and select a difficulty:
- Names **stay the same** when you restart the game
- Names **stay the same** during the entire gaming session
- Only **reset when you change level** or refresh the page

## Technical Details

### Where Team Names Appear:
1. **Score Display** - Top of game screen
2. **Arena Labels** - Left and right sides of rope
3. **Turn Indicator** - Shows current team's turn
4. **Winner Announcement** - Game over screen
5. **Final Score** - End game summary

### Character Limit:
- Maximum 20 characters per team name
- Prevents display overflow on small screens
- Ensures readability across all devices

### Text Formatting:
- Automatically converts to UPPERCASE
- Maintains consistent styling
- Professional appearance

## Troubleshooting

**Q: My team name isn't showing up**
- Make sure you entered the name BEFORE selecting difficulty
- Check that you didn't just use spaces (needs actual text)

**Q: Can I use emojis in team names?**
- Yes! Emojis work great: "🦁 LIONS vs 🐯 TIGERS"

**Q: Team name is too long and gets cut off**
- Keep names under 20 characters
- Use abbreviations if needed

**Q: How do I reset to defaults?**
- Simply leave both input fields empty
- Or refresh the page

**Q: Can I change names without losing my game?**
- No, you must finish or restart the current game
- Use "Change Level" to update names for a new game
